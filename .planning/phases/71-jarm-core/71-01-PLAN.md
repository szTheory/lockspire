<task type="auto">
  <name>Task 2: Core JARM Signer</name>
  <files>lib/lockspire/protocol/jarm.ex</files>
  <action>
    Create `lib/lockspire/protocol/jarm.ex` with a `sign(params, context)` function. 
    It must:
    1. Fetch the active signing key for the `alg` specified in context/client via `Lockspire.Storage.KeyStore.fetch_active_signing_key/1`.
    2. Decode the private JWK.
    3. Merge standard claims (`iss`, `aud`, `exp`) into the incoming `params`.
    4. Sign the resulting payload using `JOSE.JWT.sign/3` and compact it with `JOSE.JWS.compact/1`.
    Refer to PATTERNS.md analog `id_token.ex` for the exact JWT signing pattern.
    Handle missing keys gracefully by returning `{:error, :invalid_signing_key}` or similar tuple instead of crashing.
  </action>
  <verify>
    <automated>mix test test/lockspire/protocol/jarm_test.exs</automated>
  </verify>
  <done>Jarm module correctly builds and signs JWTs using :jose, enforcing the iss parameter.</done>
