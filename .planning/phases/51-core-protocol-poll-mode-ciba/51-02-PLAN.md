  <name>Task 1: Implement Backchannel Authentication Protocol</name>
  <files>lib/lockspire/protocol/backchannel_authentication.ex</files>
  <action>
    - Create `Lockspire.Protocol.BackchannelAuthentication` mirroring the `DeviceAuthorization` protocol pattern.
    - Implement `authorize/1` which:
      1. Authenticates the client using `ClientAuth`.
      2. Validates that exactly one of `login_hint`, `id_token_hint`, or `login_hint_token` is provided.
      3. Validates that the `openid` scope is present.
      4. Resolves the `subject_id` using `Lockspire.Host.AccountResolver.resolve_account/2` (per RESEARCH.md recommendation).
      5. Generates a high-entropy `auth_req_id`.
      6. Persists the authorization using `Repository.put_ciba_authorization/1`.
      7. Returns `auth_req_id`, `expires_in`, and `interval`.
  </action>
  <verify>

  </verify>
