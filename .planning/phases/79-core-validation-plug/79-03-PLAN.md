<objective>
Implement the `Lockspire.Plug.RequireToken` plug.

Purpose: This "strict enforcement" plug acts as the boundary. It inspects `conn.assigns[:access_token]` set by `VerifyToken`. If the token is missing or invalid, it immediately halts the pipeline, formats the proper `401 Unauthorized` response, and injects the RFC 6750 `WWW-Authenticate: Bearer` error headers.
Output: Working plug implementation and tests.
</objective>
